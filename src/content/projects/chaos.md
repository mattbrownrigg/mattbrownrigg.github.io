CHAOS organizes computational data across a wide system of databases while retaining 
data provenance to each institution. This pipeline ingests high-throughput DFT computations and presents novel findings from the S4E AFLOW build.

The emphasis is on large scale Machine Learning models using Graph Neural Networks, using lattice structures over a large combinatorial space.
